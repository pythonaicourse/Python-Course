# 🧠 Part 1: VS Code Setup

## 🔽 Step 1: Download VS Code

1. Open Google  
2. Search: `VS Code download`  
3. Download & install  

---

## 🔌 Step 2: Install Python Extension

1. Open VS Code  
2. Click **Extensions (left side)**  
3. Search: `Python`  
4. Click **Install**  

---

## 📄 Step 3: Create Python File

1. Go to **File → New File**  
2. Save as:
